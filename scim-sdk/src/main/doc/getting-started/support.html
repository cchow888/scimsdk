Getting Support for the UnboundID SCIM SDK

For questions, issues, and other feedback regarding the UnboundID SCIM SDK and Reference Implementation, use the forums provided at http://www.unboundid.com/labs/projects/system-for-cross-domain-identity-management/.

Note that support for the UnboundID SCIM SDK itself does not cover applications developed using the UnboundID SCIM SDK. For assistance with a third-party application developed using the UnboundID SCIM SDK, contact the organization that developed that application.
